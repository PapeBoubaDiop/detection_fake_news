À propos de l'application

Cette application utilise un modèle de machine learning pour détecter les fausses informations (fake news). Elle est conçue pour aider les utilisateurs à identifier rapidement les publications trompeuses sur Internet.

Le modèle a été entraîné sur un jeu de données contenant des centaines d'articles étiquetés comme vrais ou faux, utilisant une combinaison de techniques de traitement du langage naturel (TF-IDF, nettoyage de texte, etc.) et un classificateur comme la régression logistique ou Random Forest.

Ce projet a un but éducatif et illustratif. Il peut être affiné avec des données supplémentaires ou utilisé comme base pour une solution plus robuste.
